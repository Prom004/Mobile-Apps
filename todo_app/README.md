## 🚀 Features

- Add, remove, and display tasks
- Beautiful Material UI
- Persistent UI state
- Modular code (with pages and widgets)

## 🧰 Requirements

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (version 3.0 or later recommended)
- Dart SDK
- Android Studio or Visual Studio Code
- An emulator or physical device

## 📦 Installation & Running

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/todo_app.git
cd todo_app
````

2. **Get the Flutter dependencies:**

```bash
flutter pub get
```

3. **Run the app:**

```bash
flutter run
```

> 💡 Make sure a device or emulator is connected.

## 🛠 Common Issues

* **Dependencies not resolving?**

  ```bash
  flutter pub get
  flutter pub upgrade --major-versions
  ```

* **Emulator not detected?**
  Make sure Android Studio's AVD is running or connect a real device.
